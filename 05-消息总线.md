# ========== Spring Cloud Bus ============

可以将它理解为管理和传播所有分布式项目中的消息即可，其实本质是利用了 MQ 的广播机制在分布式的系统中传播消息，目前常用的有 Kafka 和 RabbitMQ。利用 Bus 的机制可以做很多的事情，其中配置中心客户端刷新就是典型的应用场景之一，用一张图来描述 Bus 在配置中心使用的机制
